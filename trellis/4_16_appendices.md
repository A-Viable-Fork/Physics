## Appendix B: Contract Relationship Types

**Competing:** Multiple contracts proposed for same role. One wins, others die. The function persists even when contracts fail. Example: F13 (EM coupling). Eight dead contracts, one open loop-level contract. Function survives because data demands it.

**Composing:** Multiple contracts jointly satisfy a function. Each carries a measurable fraction of the load. Losing one weakens the function proportionally. Example: F05 (neutron star support). $P_{\text{fluid}} \sim 50\%$ + $P_{\text{torsion}} \sim 50\%$. Both required for $M_{\max} = 2.323$.

**Regime-separated:** Different contracts active in different parameter regimes. The function is the classification, not either individual mechanism. Example: F06 (stellar death). Tear in thermal regime, puncture in coherent regime.

### G.1 Causality and Signaling

**Misattribution: "The state-dependent VEV permits superluminal signaling via the Gisin-Polchinski theorem."**

The reviewer argues that because the metric depends non-linearly on the quantum state (through $F_Q$), a distant measurement can instantaneously alter local geometry, enabling faster-than-light communication. The typical form of the argument invokes an EPR pair: Bob measures his qubit, "purifying" Alice's reduced state and shifting her local $F_Q$.

**Why it is wrong:** The argument confuses the *conditional* post-measurement state (which Alice can access only after receiving Bob's classical result) with the *unconditional* reduced state (which is all that's locally accessible). The no-signaling theorem states: $\text{Tr}_A[(U_A \otimes I_B)\hat{\rho}_{AB}(U_A^\dagger \otimes I_B)] = \text{Tr}_A[\hat{\rho}_{AB}] = \hat{\rho}_B$, by the cyclic property of the partial trace. This holds for *all* entanglement structures. Alice's reduced density matrix is invariant under Bob's local operations. All moments of local observables on Alice's region, including $\text{Var}(T_{\mu\nu})$ which determines the local $\mathcal{f}_Q$, are functionals of $\hat{\rho}_A$. Since $\hat{\rho}_A$ is invariant, Alice's local geometry is invariant. No signaling occurs.

The Gisin-Polchinski theorem applies when the *matter evolution itself* is non-linear ($i\hbar\partial_t|\psi\rangle = H(|\psi\rangle)|\psi\rangle$). In the present construction, matter evolves via standard linear quantum mechanics on a curved background. Only the background responds to the state, through semiclassical backreaction. This is structurally identical to standard semiclassical gravity $G_{\mu\nu} = 8\pi\langle T_{\mu\nu}\rangle_{\text{ren}}$, extended to the second moment.

**Reference:** Paper 0, Section 12.2; T2.9.

**Common variant: "But for entangled states, Bob's measurement changes Alice's reduced state."** This is the same error restated. The partial trace is invariant under Bob's local unitaries regardless of entanglement. If a reviewer insists otherwise, ask them to compute $\text{Tr}_B[(\mathbb{I}_A \otimes U_B)\rho_{AB}(\mathbb{I}_A \otimes U_B^\dagger)]$ explicitly for any bipartite state. It equals $\text{Tr}_B[\rho_{AB}]$.

**Common variant: "The construction requires a retarded causal kernel (Kaplan-Rajendran) to preserve causality."** It does not. Introducing a retarded kernel concedes a signaling problem that does not exist. The no-signaling theorem is exact for linear matter QM, regardless of how the background responds. Adding retardation would also transform the clean algebraic EFT coupling (eq. 3.10) into an integro-differential equation, changing the construction's mathematical character. Do not adopt this fix.
